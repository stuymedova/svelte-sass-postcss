# Svelte + Sass and PostCSS

Svelte app with Sass SCSS and PostCSS Autoprefixer configured.

## Overview

```
.
└── src
    ├── App.svelte
    ├── main.js
    └── styles
        ├── components
        │   ├── _button.scss
        │   ├── _mixins.scss
        │   └── _variables.scss
        ├── typography
        │   ├── _font-face.scss
        │   ├── _mixins.scss
        │   ├── _type-scale.scss
        │   └── _typography.scss
        ├── _page.scss
        ├── _reset.scss
        └── main.scss
```

## Prerequisites

[Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/)

## Setup

Run in terminal:
```
npm install
npm run dev
npm run build
```
